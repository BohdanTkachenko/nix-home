# Wishlist

Upstream features and bugs in third-party projects that affect me. Tracked
here so I can watch for resolutions and revisit my workarounds when they land.

## Ptyxis: Terminal Splits

- **Upstream**: [GNOME Design/whiteboards#279](https://gitlab.gnome.org/Teams/Design/whiteboards/-/issues/279)
- **Status**: Open (filed 2024-08-24)

Ptyxis lacks the ability to split its view into multiple vte panes
(vertically/horizontally), like terminator, tilix, or tmux. Discussion is
about whether maintainers are open to the feature being designed and
contributed.

## Ptyxis: tmux -CC integration

- **Upstream**: [chergert/ptyxis#70](https://gitlab.gnome.org/chergert/ptyxis/-/issues/70)
- **Status**: Closed (filed 2024-01-10)

Request to add `tmux -CC` control-mode integration to Ptyxis, like iTerm2
provides on macOS. Would let tmux windows/panes be rendered as native
terminal tabs/splits instead of in-terminal ASCII.
